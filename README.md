# Article_PHP
W aplikacji istnieją takie funkcje jak:
  -Logowanie
  -Rejestracja
  -Dodwanie, Edytowanie, Usuwanie artykułu
  -Wyświetlenie listy artykułów.
  -Wyświetlenie całości wybranego artykułu.
